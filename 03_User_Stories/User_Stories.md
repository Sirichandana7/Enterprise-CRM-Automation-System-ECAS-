# User Stories

## User Story 1

As a Sales Executive,

I want new Leads assigned automatically,

So that I can contact customers quickly.

---

## User Story 2

As a Sales Manager,

I want to monitor the Sales Pipeline,

So that I can forecast revenue.

---

## User Story 3

As a Support Executive,

I want Cases assigned automatically,

So that customer issues are resolved quickly.

---

## User Story 4

As a CEO,

I want executive dashboards,

So that I can monitor company performance.
