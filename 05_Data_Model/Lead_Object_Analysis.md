# Lead Object Analysis

## Standard Object

Lead

## Purpose

Stores information about prospective customers before they become actual customers.

## Standard Features Reviewed

- Fields & Relationships
- Page Layout
- Buttons
- Standard Actions

## Observations

The Lead object already contains standard fields such as Company, Last Name, Phone, Email, Lead Source, and Status. These fields satisfy many business requirements, but additional custom fields may be needed based on Syntech Solutions' processes.
